Проброс портов

Привязка портов контейнера к портам хоста. Формат: host:container/proto. Изменения применяются после «Пересоздать».

Переменные окружения

Будут переданы в контейнер как env-переменные. Используются для токенов, ключей API и т.п. Применяются после «Пересоздать».

GitHub автообновление

Если включено, перед стартом/рестартом панель сама обновит код через git clone или git pull. Если Git URL заполнен, автообновление включится при сохранении. Папка должна быть пустой либо уже быть git-репозиторием без локальных изменений.
Лимиты применяются после «Пересоздать».
